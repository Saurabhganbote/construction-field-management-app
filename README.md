# Construction Field Management App

This project is developed as part of the **Frontend Web Developer Intern Selection Task**.

The application demonstrates a simple Construction Field Management interface where a user can log in, view project details, and submit a Daily Progress Report (DPR).

---

## Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap

---

## Features Implemented

1. Login Screen  
   - Email and Password fields  
   - Mock authentication  
   - Error message for invalid login  

2. Project Dashboard  
   - Card-based UI displaying construction projects  
   - Project Name, Status and Location  

3. DPR Form  
   - Date selection  
   - Weather dropdown (Sunny / Cloudy / Rainy)  
   - Work description field  
   - Worker count input  
   - Image upload with preview  

4. Responsive Design  
   - Works on mobile, tablet and desktop  
   - Implemented using Bootstrap grid system
